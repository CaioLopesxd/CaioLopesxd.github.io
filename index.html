Sobre

Olá, meu nome é Caio Dos Santos Lopes, tenho 20 anos e estou em busca da minha primeira experiência na área. Minha jornada como desenvolvedor começou em 2022, quando fiz um curso profissionalizante de WebDesign. Nesse curso, tive contato com conceitos básicos de lógica de programação e um pouco de desenvolvimento backend com PHP. Atualmente, estou estudando Python, PHP, C++ e desenvolvimento mobile com ReactJS na UniSatc.

# CRUD de Usuários com PHP

Um sistema de cadastro de usuários feito com PHP e MySQL. O objetivo desse projeto era praticar com PHP e MySQL com a criação de um CRUD. Futuramente, pretendo melhorá-lo e adicionar novas funcionalidades.

# Service Order Manager

Um mini ERP criado durante o curso de React Native na UniSatc. O objetivo desse projeto era praticar com React Native e Firebase. É um sistema de gerenciamento de ordens de serviço, onde o usuário pode criar, editar suas ordens de serviço e anexar seus clientes a elas.

# Python BackEnd

Um blog feito com Python e Flask. O objetivo desse projeto era praticar com Python , Flask , MongoDB e Bootstrap. Nesse projeto você pode criar posts, deletar e visualizar todos os posts de qualquer autor, pretendo futuramente criar algo semelhante a uma rede social, onde só aparece os itens das pessoas que segue.
